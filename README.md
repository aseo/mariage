# Mariage

AI-powered Korean food and drink pairing recommendations.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your Gemini API key to `.env.local`:
```
GOOGLE_AI_API_KEY=your_key_here
```

3. Run the development server:
```bash
npm run dev
```

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Google Gemini API
